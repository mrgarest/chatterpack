# Twitch Chatterpack Extension

**Chatterpack** is a browser extension for Twitch that enhances chat interaction by providing convenient tools for automating messages using custom commands, flexible moderation based on keywords and phrases, and personalized highlighting of important messages and users.

⚠️ If you use other Twitch chat extensions, there may be conflicts between them. Currently, **Chatterpack** is compatible with **7TV**.

## Key features:

### Commands

Adding custom commands to send pre-written messages.

![Commands](https://i.ibb.co/214rV5rT/3.gif)

### Moderation

Adding a rule for automatic chat moderation based on keywords and phrases.

![Moderation](https://i.ibb.co/mCf8BvHC/moderation.gif)

### Highlight

Highlight specific messages or users in chat so they never get lost in the stream.

![Highlight](https://i.ibb.co/nqYd64q4/highlight.gif)
